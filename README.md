## 📳 Вкладка «Тряска»

1. **Детектор тряски** — акселерометр + настраиваемый порог 5–30 м/с²
2. **Выбор MP3** — ввод пути вручную или через файловый менеджер
3. **ProgressBar воспроизведения** — горит ровно пока играет звук
4. **Счётчик трясок** + время последней
5. **Вибрация** при тряске (паттерн 0–100–50–100 мс)
6. **Foreground Service** — работает при свёрнутом приложении и после свайпа из recents
7. **WakeLock** — CPU не засыпает
8. **Кнопка Stop** прямо в уведомлении
9. **Виджет рабочего стола** — быстрый старт/стоп без открытия приложения

## 🎮 Вкладка «Игры»

### 🎲 Кубик
- Тряска или тап → бросок кубика 1–6
- 3D-анимация: scale + rotationY
- Статистика: среднее, частота выпадений, последние броски

### 🎰 Рулетка
- Европейская рулетка (0–36)
- Ставки: красное/чёрное, чётное/нечётное, конкретное число
- Тряска = крутить колесо
- Баланс 1000 💰, при банкротстве — автосброс

### 🔢 Угадайка
- Телефон загадывает число 1–100
- Тряска = автоматически угадываем середину (бинарный поиск)
- Счётчик трясок до победы

### 🏎 Гонки
- ASCII-гонки: машинка едет по дороге, объезжает препятствия (#)
- Тряска = буст скорости (накапливается до ×15)
- 3 жизни, счётчик очков

### 🐍 Змейка
- Полноценная реализация на Canvas
- 20×20 клеток, ~60 fps
- Кнопки ↑↓←→ на экране

### 🔢 2048
- Полная реализация на Canvas (свайп пальцем)
- Цветовая схема оригинала
- Определение победы (2048) и поражения (нет ходов)

## 📡 Вкладка «Инфо»

### 🎙 Шумомер
- Измеряет уровень шума в **дБ** через микрофон
- Обновление каждые 200 мс
- Цветной индикатор (зелёный → жёлтый → оранжевый → красный)
- Фиксация максимума

### 📡 Wi-Fi сканер
- Список всех видимых сетей
- Сигнал в % и dBm, частота (2.4/5 GHz)
- Иконки 🔒/🔓 безопасности
- Сортировка по силе сигнала

### 🌦 Погода
- OpenWeatherMap API (бесплатный ключ)
- Температура, ощущается, влажность, ветер, описание
- Иконки погоды через эмодзи

### 💱 Курс валют
- ЦБ РФ JSON API (без ключа)
- USD, EUR, GBP, CNY, JPY, CHF и другие
- Стрелки ▲▼ изменения к предыдущему дню

### 🌐 Мой IP
- Внешний IP через `ipinfo.io`
- Страна, город, провайдер, часовой пояс, координаты
- Флаг страны (emoji)

### 💀 Матрица
- Анимация падающих японских символов (ｦｧｨ...) + 0–9 + A–F
- Индивидуальная скорость и яркость каждой колонки
- ~25 fps, белая «голова» и зелёный «хвост»

## 🎵 Вкладка «Медиа»

### 🎵 MP3 Плеер
- Плейлист из нескольких файлов
- ▶ ⏸ ⏹ ⏮ ⏭ управление
- SeekBar прогресса с перемоткой
- **Тряска = следующий трек**

### 🎙 Диктофон
- Запись в AAC (M4A), 44100 Hz, 128 kbps
- Таймер записи + уровень сигнала в реальном времени
- Список записей в кэше, тап = воспроизвести

### ✂️ Создание рингтонов
- Выбор аудио файла
- Задать начало и конец (секунды)
- Предпрослушать отрезок
- Обрезать и **установить как рингтон системы**

### 📷 OCR (распознавание текста)
- ML Kit Text Recognition (работает офлайн)
- Выбор из галереи или съёмка прямо в приложении
- Статистика: блоков, слов, символов
- Кнопка «Скопировать текст»

### 💬 ChatGPT
- Полноценный multi-turn чат с GPT-3.5-turbo
- Пользователь вводит свой API ключ (хранится в SharedPreferences)
- История чата в памяти сессии
- Кнопка очистки истории

## ⚙️ Вкладка «Ещё»

### 🔦 Стробоскоп
- Мигает вспышкой камеры с частотой 1–20 Hz
- SeekBar частоты
- **Тряска = ускоряет стробоскоп** (+3 Hz за тряску)

### 📍 Компас + Уровень
- Компас: векторная стрелка на Canvas, буквы СВЮЗ
- Пузырьковый уровень рядом
- Оба работают от магнитометра + акселерометра

### ⏱ Секундомер
- Точность до сотых секунды (~60 fps)
- Фиксация кругов: 🥇 лучший, 🐢 худший
- **Тряска = новый круг**
- Пауза, продолжить, сброс

### 🔤 Морзянка
- Короткая тряска (<300 мс) → точка ·
- Длинная тряска (>300 мс) → тире −
- Пауза 1.5 сек → конец буквы (декодируем)
- Пауза 3 сек → пробел
- Таблица: кириллица + латиница + цифры

### 🐾 Тамагочи
- Виртуальная собачка 🐶
- **Тряска = кормить**, тап = гладить, кнопка = играть
- Голод и счастье снижаются каждые 30 сек
- Смерть при голоде=0 и счастье<10, кнопка «Возродить»

### 📱 QR
- **Генератор**: любой текст/URL → QR через ZXing
- **Сканер**: ZXing IntentIntegrator (встроенный сканер)
- Кнопка копирования результата

### 🆘 SOS
- Ввод номера экстренного контакта
- Активация через переключатель
- **3 тряски за 5 секунд → SMS с координатами GPS**
- Формат: «ЭКСТРЕННО! Мне нужна помощь! Координаты: lat, lon»

### ⚙️ Настройки
- Тема: Светлая / Тёмная / AMOLED
- Сохранение API ключей ChatGPT и OpenWeatherMap

## ⚠️ Предупреждения

- **Батарея**: WakeLock + акселерометр 50Hz работают постоянно. При длительном использовании — заряжайте телефон.
- **Стробоскоп**: может вызвать дискомфорт при частоте >15 Hz. Не используйте при эпилепсии.
- **SOS**: протестируйте через кнопку «Тест» перед реальным использованием. SMS стоят денег.
- **Firebase**: замените `google-services.json` своим файлом из Firebase Console.
